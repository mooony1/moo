# Conversion Optimization Checklist

A practical checklist for improving your e-commerce conversion rate.

## Product Pages

- [ ] High-quality images (minimum 3 per product)
- [ ] Zoom functionality on images
- [ ] Clear pricing with no hidden costs
- [ ] Stock availability displayed
- [ ] Estimated delivery date shown
- [ ] Trust badges near buy button
- [ ] Customer reviews visible
- [ ] Size guide (if applicable)
- [ ] Clear return policy link

## Checkout Flow

- [ ] Guest checkout available
- [ ] Progress indicator visible
- [ ] Form fields minimized
- [ ] Multiple payment options
- [ ] Security badges displayed
- [ ] Order summary visible throughout
- [ ] Shipping costs shown early
- [ ] Promo code field not too prominent

## Mobile Experience

- [ ] Thumb-friendly button placement
- [ ] Sticky add-to-cart button
- [ ] Autofill enabled for forms
- [ ] Fast page load (under 3 seconds)
- [ ] Easy navigation back to shopping

## Trust Signals

- [ ] SSL certificate active
- [ ] Contact information visible
- [ ] Live chat or quick support option
- [ ] Money-back guarantee displayed
- [ ] Real customer reviews (not just 5 stars)

## Cart Abandonment

- [ ] Exit intent popup configured
- [ ] Abandoned cart email sequence set up
- [ ] Cart contents saved for returning visitors
- [ ] Clear cart editing options
